E-commerce con Vanilla JS

El equipo de una conocida empresa me ha puesto a disposición una API, la cual permite acceder a un conjunto de métodos orientados a facilitar la integración, desde sistemas externos hacia la compañía.

Estos métodos permitirán obtener información desde un servidor externo. Así, por ejemplo, obtener productos y datos para su usabilidad.

Esta API permite llamadas del tipo REST y utiliza JSON para el envío y recepción de información.

Esta experiencia me permitió:
  • Implementar mi pensamiento lógico
  • Organizar mis prioridades en un corto plazo de tiempo
  • Documentar agilmente los features y endpoints
Tecnologías y Herramientas

Front: VanillaJS, HTML, CSS

Back: Node.js, Express.js, Sequelize, MySQL, SQL, MVP

Herramientas: Git, GitHub, Lucidchart

Funcionalidades
  • Ver todos los productos
  • Buscar productos por nombre
  • Categorizar productos
  • Ver productos de acuerdo a su categoría
