The Movie Code

Desarrollo React/Node y Diseño Web

Como desarrolladora Full Stack y Diseñadora Web, tuve la oportunidad de participar en un proyecto significativo durante mi participación en un Coding Bootcamp. En este proyecto, me enfoqué en la creación una aplicación web innovadora que permitía a los usuarios explorar películas, utilizando la API de TMDB como fuente de datos.

Esta Experiencia Me Proporcionó:

1. Habilidades de Resolución de Problemas En Tecnología:
El proceso de desarrollo de la aplicación me desafió a abordar problemas complejos y a encontrar soluciones efectivas.

2. Agilidad en la Adopción de Nuevas Tecnologías:
Trabajé con una gama de tecnologías, lo que me permitió aprender y aplicar rápidamente nuevas herramientas según las necesidades del proyecto.

3. Ciclo Completo de Desarrollo:
Desde el diseño hasta el despliegue, tuve un enfoque completo en el ciclo de vida del desarrollo de software, demostrando habilidades en diseño de interfaz, programación y configuración de servidores.

Tecnologías Relevantes:

- Front-End:
Utilicé React, Redux y CSS para una interfaz de usuario atractiva, receptiva y funcional.
- Back-End:
Implementé el back utilizando Node.js, Express.js, Sequelize en conjunto PostgreSQL para gestionar la base de datos.
- Herramientas:
Utilicé Git y GitHub para el control de versiones, y Figma para el diseño y la planificación de la interfaz.
Funcionalidades A Destacar:

Experiencia de Usuario Segura y Personalizada:
Implementación de autenticación de usuarios, incluyendo la creación de perfiles y el inicio de sesión seguro.

Explorar Perfiles:
Facilidad de exploración de perfiles de otros usuarios y visualización de sus selecciones favoritas.

Sesiones Continuas e Ininterrumpidas:
Mantenimiento de la sesión del usuario incluso después de acciones como cerrar el navegador o actualizar la página.
